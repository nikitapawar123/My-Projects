Barah Khadi - बारह खडी

For the pronounciation of any sentences we must need vovel and consonant, for the best understanding see the below tables.

कोणत्याही वाक्याच्या उच्चारासाठी आपल्याला स्वर आणि व्यंजन आवश्यक आहेत, उत्तम समजण्यासाठी खालील तक्ते पहा.

VOVEL'S TABLE - स्वर सारणी
अ = a आ = a इ = i ई = ee उ = u ऊ = oo
ए = e ऐ = ai ओ = o औ = au अं = am अः = ah

CONSONANTS TABLE - व्यंजन
क् =k ख् =kh ग् =g घ् =gh च् =ch छ् =chh
ज् =j झ् =jh ट् =t ठ् =th ड् =d ढ् =dh
ण =n त् =t थ् =th द् =d ध् =dh न् =n
प् =p फ् =ph ब् =b भ् =bh म् =m य् =y
र् =r ल् =l व् =w/v श् =sh ष् =sh स् =s
ह् =h क्ष् =ksh ज्ञ् =gy

Above hindi words are incompleted word because there are not using any vovel, when we add the vovel on the that word it makes complet word.

वरील मराठी शब्द अपूर्ण शब्द आहेत कारण तेथे कोणतेही स्वर वापरले जात नाहीत, जेव्हा आपण त्या शब्दावर स्वर जोडतो तेव्हा तो पूर्ण शब्द बनतो.

ज्याप्रमाणे 'क्' ला 'अ' ला जोडल्यास 'क' तयार होतो, त्याचप्रमाणे आपण खालील उदाहरणावरून समजू शकतो.

k(क्)+a(अ)= ka(क)
kh(ख्)+a(अ) = kha(ख)
------------------
-------------------

For creating an spacelling of the word like ka, kaa, ki, kee .........etc follow the above table.

उक्त टेबलचा उपयोग करून आपण क, का, कि, की ----------------- इत्यादींची स्पेलिंग तयार करू शकतो.
